# Remote Zugriff
[← Zur Startseite](index.md)

Remote Zugriff ist ein Werkzeug zur Hilfeleistung.
Er erfolgt nur auf Einladung, zeitlich begrenzt, und transparent.

## Grundsätze
- Kein Dauerzugriff.
- Keine Hintertüren.
- Keine Überwachung.
- Keine stillschweigende Automatisierung.

## Ablauf
1. Termin vereinbaren.
2. Remote Zugriff nur für die Dauer der Sitzung.
3. Nach Abschluss wird die Verbindung beendet.
4. Wenn sinnvoll, gibt es eine kurze Dokumentation der Änderungen.

## Tools
In der Regel AnyDesk, wenn es passt.
Alternativen sind möglich, wenn sie für dich einfacher sind.
